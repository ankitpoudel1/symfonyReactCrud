<h4>Requirements:</h1>

node >= v10.24.1

symfony cli

if you don't have symfony cli installed, install it by running the following command

```
wget https://get.symfony.com/cli/installer -O - | bash
```

<h4>default ports:</h4>
frontend 3000
backend 80000


<h4>Run the development servers</h4>

```
sh ./start.sh
```
